# 01 — Instalación

> Contenido completo en `EMPIEZA-AQUI.md` y el comando `/setup`.

## Requisitos

- Node.js 22 (descarga en nodejs.org)
- VS Code con extensión Claude Code
- Cuenta en OpenRouter con crédito

## Instalación rápida

```bash
# Opción A: con Claude Code (recomendado)
# Abre la carpeta en VS Code y escribe /setup

# Opción B: wizard de consola
npm run wizard

# Opción C: manual
npm install
npm run build
```

## Verificar instalación

```bash
npm run check
```

Debe mostrar 7 checks en verde.
